## Olá, eu sou o Paulo 👋

Desenvolvedor **Python** com foco em **backend, integração de dados e sistemas em tempo real** aplicados à área da saúde.

Trabalho desenvolvendo aplicações que resolvem problemas operacionais concretos: painéis assistenciais exibidos em TVs pelas unidades de internação, UTIs, Pronto Socorro e Centro Cirúrgico; sistemas internos de controle e auditoria; e ferramentas que substituíram planilhas por aplicações web multiusuário.

🔹 **APIs e aplicações web** com Python e Flask — arquitetura em camadas e autenticação via Active Directory  
🔹 **Otimização de consultas SQL** — reescrita de queries em produção com alta performance e redução de custos  
🔹 **Integração com bases corporativas** — SQL Server, SQLite (WAL, FTS5), Active Directory e DATASUS  
🔹 **Migração de BI para aplicações autônomas** — medidas DAX traduzidas para SQL e Python, atualizadas em tempo real  
🔹 **ETL e dados públicos de saúde** — importador dinâmico de schemas baseado em layouts oficiais do DATASUS  

💡 Penso sistemas para **produção**: cache com TTL para proteger o banco, fallback estático para alta disponibilidade, trilha de auditoria e privacidade por padrão.
