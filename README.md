## Olá, eu sou o Paulo 👋

Desenvolvedor **Python** com foco em **backend, integração de dados e sistemas em tempo real** aplicados à área da saúde.

Trabalho no Hospital Memorial desenvolvendo aplicações que resolvem problemas operacionais concretos: painéis assistenciais exibidos em TVs pelas unidades de internação, UTIs, Pronto Socorro e Centro Cirúrgico; sistemas internos de controle e auditoria; e ferramentas que substituíram planilhas por aplicações web multiusuário.

🔹 **APIs e aplicações web** com Python e Flask — arquitetura em camadas, APIs REST e autenticação corporativa via Active Directory
🔹 **Otimização de consultas SQL** — já reescrevi queries que travavam sistemas em produção, com resultado idêntico e uma fração do custo
🔹 **Integração com bases corporativas** — SQL Server, SQLite (WAL, FTS5), Active Directory e o FTP público do DATASUS
🔹 **Migração de BI para aplicações autônomas** — medidas DAX traduzidas para SQL e Python, atualizadas em tempo real
🔹 **ETL e dados públicos de saúde** — importador que lê os layouts oficiais do DATASUS e monta o schema dinamicamente

💡 Penso sistemas para **produção**: cache com TTL para proteger o banco, fallback estático para nunca exibir erro em uma TV de corredor, trilha de auditoria em toda movimentação sensível e privacidade por padrão.
